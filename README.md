📅 AUTOMAÇÃO: Gestão de Reuniões Teams via Excel/VBA
  Automação em Excel/VBA para agendamento de reuniões em massa no Microsoft Teams.

⚡ O que faz
  Criação em massa: Agenda múltiplas reuniões via planilha.
  Links Automáticos: Gera o link do Teams para cada evento.
  Envio de Convites: Dispara e-mails automaticamente para os participantes.
  Cancelamento: Permite o cancelamento em lote das reuniões criadas.

🛠️ Tecnologias
  Excel + VBA
  Integração com Outlook/Teams

🚀 Como usar
  Preencha a planilha com os dados das reuniões.
  Clique no botão de execução (Macro).
  Os convites serão enviados via Outlook.
